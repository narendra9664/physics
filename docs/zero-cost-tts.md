# Zero-Cost TTS — Locked Configuration

## Primary Voice

**en-US-DavisNeural** — deep, gravelly, authoritative storyteller.

## Settings

```python
VOICE = "en-US-DavisNeural"
RATE  = "-8%"    # slightly slower for measured storytelling
PITCH = "-3Hz"   # slightly deeper for gravitas
```

## Why DavisNeural?

- Deep, warm, authoritative tone
- Sounds like a documentary narrator
- Good for physics content where credibility matters
- Free via edge-tts library (uses Microsoft Edge Read Aloud API)

## Alternative Voices (tested)

| Voice | Style | When to use |
|-------|-------|-------------|
| en-US-GuyNeural | Expressive, passionate | If DavisNeural feels too flat for a specific topic |
| en-US-AndrewNeural | Warm, casual | If you want a more conversational feel |
| en-GB-RyanNeural | British, professional | If the topic suits a British narrator |

## Technical Notes

- edge-tts requires internet (uses Microsoft's servers)
- Does NOT support SSML style tags (narration, storytelling, etc.)
- Workaround: use `rate` and `pitch` parameters for tone control
- Generates word-level timestamps for animation sync

## Integration

Voiceover is generated by `src/generate_voiceover.py`:
```bash
python src/generate_voiceover.py --episode 001-gravity-spacetime
```

Output:
- `episodes/{slug}/voiceover.mp3`
- `episodes/{slug}/timing.json` (word-level timestamps)
